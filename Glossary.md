# Glossary

- constant ค่าคงที่
- variable ตัวแปร
- array อาเรย์
- dictionary ดิกชันนารี
- set เซท
- property พร็อพเพอร์ตี้
- class คลาส
- method เมธอด
- function ฟังก์ชัน
- parameter พารามิเตอร์
- argument อากิวเมนต์
- object oriented programming การเขียนโปรแกรมเชิงวัตถุ
