เกี่ยวกับ Swift

Swift คือภาษาที่ถูกพัฒนาขึ้นมาใหม่เพื่อใช้ในการพัฒนาแอปพลิเคชั่นบน iOS, macOS, watchOS และ tvOS ภาษาสวิฟต์ได้รวมเอาข้อดีต่างๆของภาษา C กับ Objective-C มาไว้ด้วยกัน และทำงานร่วมกับภาษา C โดยไม่มีข้อจำกัด ภาษาสวิฟต์มีรูปแบบการเขียนโปรแกรมที่ปลอดภัย และยังได้เพิ่มคุณสมบัติต่างๆที่มีในภาษาสมัยใหม่ ซึ่งจะทำให้การเขียนโปรแกรมเป็นเรื่องง่าย มีความยืดหยุ่น และสนุกมากขึ้นกว่าแต่ก่อน สวิฟต์ถูกพัฒนาขึ้นมาใหม่ทั้งหมด แต่ยังคงใช้งานเฟรมเวิร์ค Cocoa และ Cocoatouch ได้เช่นเดิม

Swift ใช้เวลาในการพัฒนาขึ้นมาหลายปี แอปเปิ้ลได้วางรากฐานของภาษาด้วยการปรับปรุงสิ่งต่างๆที่มีอยู่แล้วเช่น คอมไพเลอร์ ดีบักเกอร์ รวมไปถึงเฟรมเวิร์คพื้นฐาน เราได้ทำให้การจัดการความจำง่ายขึ้นด้วย Automatic Reference Counting (ARC) และในส่วนของเฟรมเวิร์คต่างๆของภาษา ก็ได้พัฒนาขึ้นมาจากพื้นฐานที่แข็งแกร่งของ Cocoa ซึ่งได้รับการปรับปรุงให้มีมาตรฐานและทันสมัยมากขึ้น ต้องขอบคุณภาษา Objective-C ที่ได้รองรับเทคโนโลยีของภาษายุคใหม่อย่าง block, collection literals, module, framework ทำให้เราได้นำสิ่งต่างๆเหล่านี้มาประยุกต์ใช้กับภาษา Swift ได้อย่างไม่ติดขัด และทำให้สวิฟต์เป็นภาษาใหม่สำหรับการพัฒนาซอฟแวร์ในอนาคตของแอปเปิล

Swift จะทำให้นักพัฒนา Objective-C รู้สึกคุ้นเคย เพราะได้นำเอาประเภทข้อมูลที่ยืดหยุ่น อีกทั้งการตั้งชื่อพารามิเตอร์ที่อ่านง่ายของภาษา Objective-C เข้ามาใช้งาน ภาษาสวิฟต์ทำงานร่วมกับเฟรมเวิร์ค Cocoa ได้อย่างราบรื่น สามารถนำโค้ดของ Objective-C มาใช้ร่วมกับ Swift ได้อีกด้วย จากสิ่งเหล่านี้เอง ทำให้ Swift มีฟีเจอร์ใหม่ๆ และรวมเอาวิธีการเขียนโปรแกรมเชิงกระบวนการกับการเขียนโปรแกรมเชิงวัตถุ เข้ามาไว้ด้วยกันได้

Swift เป็นกับมิตรโปรแกรมเมอร์มือใหม่ เป็นภาษาที่ใช้งานง่ายเหมือนกับภาษาสคริป แต่มีประสิทธิภาพสูง สวิฟต์มีฟีเจอร์ playgrounds ซึ่งเป็นที่ให้โปรแกรมเมอร์ได้ทดลองเขียนโค้ดและเห็นผลลัพธ์ในทันที โดยที่ไม่ต้องยุ่งยากในการสร้างแอปพลิเคชั่นและสั่งให้ทำงาน

Swift ได้รวมเอาสิ่งที่ดีที่สุดของภาษาสมัยใหม่ และผ่านการคิดและกรั่นกรองโดยวิศกรของแอปเปิ้ล ความง่ายของภาษาในการพัฒนา และประสิทธิภาพของคอมไพเลอร์ได้ถูกปรับปรุงไปพร้อมกันโดยไม่สูญเสียสิ่งใดสิ่งหนึ่งไป มันได้ถูกออกแบบให้เหมาะสมกับโปรแกรมเล็กๆอย่าง "hello, world" ไปจนถึงกระทั่งระบบปฎิบัติการ ทั้งหมดนี้ทำให้ Swift คือการลงทุนเพื่ออนาคตของนักพัฒนาและแอปเปิ้ล

Swift คือวิธีการในการเขียนแอปพลิเคชัน iOS, macOS , watchOS, tvOS ที่วิเศษที่สุด และมันจะยังพัฒนาต่อไปพร้อมกับฟีเจอร์และความสามารถใหม่ๆ จุดมุงหมายของเราคือทำให้ Swift เป็นสิ่งที่ยอดเยี่ยมที่สุด เราอดใจรอไม่ไหวที่จะได้เห็นคุณสร้างสรรสิ่งต่างๆขึ้นมาด้วยภาษาสวิฟต์
