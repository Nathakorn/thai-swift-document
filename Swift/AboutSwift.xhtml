เกี่ยวกับ Swift

Swift คือภาษาใหม่สำหรับการพัฒนาแอปพลิเคชั่น iOS, macOS, watchOS และ tvOS ซึ่งได้รวมเอาข้อดีของภาษา C และ Objective-C เข้าไว้ด้วยกัน และยังสามารถทำงานร่วมกันกับภาษา C ได้อีกด้วย ภาษา Swift มีรูปแบบการเขียนโปรแกรมที่ปลอดภัยและยังได้เพิ่มคุณสมบัติต่างๆของภาษาสมัยใหม่ ซึ่งจะทำให้การเขียนโปรแกรมนั้นเป็นเรื่องง่าย มีความยืดหยุ่น และสนุกสนานมากขึ้นกว่าแต่ก่อน ภาษา Swift ได้รับการพัฒนาขึ้นมาใหม่ทั้งหมด แต่ยังใช้งาน Cocoa และ Cocotouch Framework ได้เช่นเดิม

Swift ได้ใช้เวลาในการพัฒนาขึ้นมาหลายปี ซึ่งแอปเปิ้ลได้วางโครงสร้างพื้นฐานของภาษาด้วยการปรับปรุงสิ่งที่มีอยู่แล้วเช่น คอมไพเลอร์ ดีบักเกอร์ เฟรมเวิร์ค และทำให้การจัดการหน่วยความจำง่ายขึ้นด้วย Automatic Reference Counting (ARC) ในส่วนเฟรมเวิร์คต่างๆก็ได้สร้างขึ้นจากพื้นฐานที่แข็งแกร่งของ Cocoa ที่ได้รับการปรับปรุงให้ทันสมัยและมีมาตรฐาน ต้องขอบคุณภาษา Objective-C ที่ได้การพัฒนาและปรับปรุงให้รองรับ block, collection literals, module, framework และปรับปรุงให้มีคุณสมบัติของภาษาสมัยใหม่และกลายมาเป็นรากฐานของภาษา Swift และทำให้ Swift คืออนาคตของการพัฒนาซอฟแวร์

Swift จะให้ทำให้นักพัฒนารู้สึกคุ้นเคยเหมือนการใช้ Objective-C ทั้งโค้ดที่อ่านง่าย ความยืดหยุ่นของชนิดข้อมูล การตั้งชื่อพารามิเตอร์ และยังทำงานร่วมกันกับ Cocoa Framework ได้อย่างราบรื่น โค้ดของภาษา Swift และ Objective-C สามารถนำมาใช้งานด้วยกันได้ สิ่งเหล่านี้ทำให้ Swift สามารถนำเอาฟีเจอร์ใหม่ๆ และรวมเอาการเขียนโปรแกรมเชิงกระบวนการและการเขียนโปรแกรมเชิงวัตถุเข้าไว้ด้วยกันได้

Swift เป็นกับมิตรโปรแกรมเมอร์มือใหม่ เป็นภาษาที่ใช้งานง่ายเหมือนกับภาษาสคริป แต่มีประสิทธิภาพสูง มันได้รองรับ playgrounds ซึ่งเป็นฟีเจอร์ที่อนุญาติให้โปรแกรมเมอร์ทดลองเขียนโค้ดและเห็นผลลัพธ์ในทันที โดยที่ไม่ต้องยุ่งยากในการสร้างแอปพลิเคชั่นและสั่งให้ทำงาน

Swift ได้รวมเอาสิ่งที่ดีที่สุดของภาษาสมัยใหม่ ซึ่งได้ผ่านการคิดและกรั่นกรองโดยวิศกรในแอปเปิ้ล ความง่ายในการพัฒนา และประสิทธิภาพของคอมไพเลอร์ได้ถูกปรับปรุงไปพร้อมกันโดยไม่สูญเสียสิ่งใดสิ่งหนึ่งไป มันได้ถูกออกแบบให้เหมาะสมกับโปรแกรมเล็กๆอย่าง "hello, world" ไปจนถึงกระทั่งระบบปฎิบัติการ ทั้งหมดนี้ทำให้ Swift คือการลงทุนเพื่ออนาคตของนักพัฒนาและแอปเปิ้ล

Swift คือวิธีการในการเขียนแอปพลิเคชัน iOS, macOS , watchOS, tvOS ที่วิเศษที่สุด และมันจะยังพัฒนาต่อไปพร้อมกับฟีเจอร์และความสามารถใหม่ๆ จุดมุงหมายของเราคือทำให้ Swift เป็นสิ่งที่ยอดเยี่ยมที่สุด เราอดใจรอไม่ไหวที่จะเห็นคุณได้สร้างสรรสิ่งต่างๆขึ้นมา
